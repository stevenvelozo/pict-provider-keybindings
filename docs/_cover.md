# Pict-Provider-Keybindings

> Centralized keyboard shortcuts for Pict applications — register once, document automatically

- One registry for app-global and per-view bindings
- Chords, key sequences, and a platform-aware Mod (⌘ / Ctrl) abstraction
- Built-in, themeable cheatsheet overlay — press ?
- Lifecycle-friendly: views register in onAfterRender, bindings auto-evict on unmount
- Input/IME guarding, suspend() / resume(), scopes, and per-binding disable
